# IR Sensor Object Counter using Arduino

This is a real-time mini project developed using Arduino UNO and an IR sensor to automatically count objects.

## Description
The system detects objects using an IR sensor and increments the count using Arduino logic. The count is displayed on a 16x2 LCD display.

## Components Used
- Arduino UNO
- IR Sensor
- 16x2 LCD
- I2C module
- Jumper wires
